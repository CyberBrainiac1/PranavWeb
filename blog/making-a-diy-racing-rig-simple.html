Making a DIY Racing Rig (Simple)

diy simracing simple build

Goal

I wanted a racing wheel and pedal setup without the price tag. This version is the simplest working build: one steering axis, one pedal axis, and a tiny microcontroller that shows up as a USB gamepad.

Parts List (Minimal)

  • Microcontroller: Any HID-capable board (XIAO RP2040 or similar)
  • Steering sensor: Magnetic encoder or a cheap potentiometer
  • Pedal sensor: Potentiometer or rotary encoder
  • Hardware: Basic brackets, bolts, and a return spring
  • Wiring: Jumper wires + a small breadboard

Simple Build Steps

  1. Mount the steering sensor so it rotates with the wheel shaft.
  2. Mount the pedal sensor so it moves as you press the pedal.
  3. Wire both sensors to the microcontroller (power, ground, signal).
  4. Flash firmware that maps sensor values to two USB axes.
  5. Open the Windows controller tester and calibrate center points.

What Makes It "Good"

  • Stable center: The wheel always returns to a clear center point.
  • Smooth input: Axis values move steadily, not in jumps.
  • Clean wiring: Short wires, shared ground, and no loose connections.
  • Upgradeable: You can swap in better sensors later.
